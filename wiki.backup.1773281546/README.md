# Casita Miro Marketing POC - Wiki

This is the **local wiki** (Docsify) for the Casita Miro Marketing Proof of Concept. 

## Quick Start

1. **View locally:** Open `index.html` in a browser
2. **Navigate:** Use sidebar menu or click links throughout
3. **Start here:** Read [home.md](home.md) or [executive-summary.md](executive-summary.md)

## Structure

- **`index.html`** — Docsify configuration and entry point
- **`_sidebar.md`** — Navigation menu
- **`home.md`** — Welcome and overview
- **`executive-summary.md`** — 5-minute summary
- **`strategy/`** — As-Is, SWOT, Business Strategy
- **`marketing/`** — Marketing Plan (website, social, content, etc.)
- **`tactics/`** — Social calendar, email templates, blog topics, KPIs
- **`reference/`** — House of Eros and Movida references

### Publishing to Vercel
This wiki is ready to be published to Vercel as a static site (like House of Eros).

**To deploy:**
1. Configure Vercel project pointing to `/wiki/` folder
2. Set root to `wiki/` (not project root)
3. No build command needed (Docsify is client-side)
4. Deploy and share URL
